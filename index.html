Практическое задание: Сайт-визитка с формой регистрации

Локальный сайт для отработки навыков HTML, CSS и JavaScript

Описание проекта

Это учебный проект, созданный для практики работы с веб-технологиями. Сайт работает полностью локально без необходимости хостинга.

Функциональность:

  • Форма "регистрации" с валидацией
  • Локальное сохранение данных в браузере
  • Таблица с зарегистрированными пользователями
  • Адаптивный дизайн
  • Поддержка HTTPS протокола
  • Настройки почтовых протоколов

Используемые технологии:

HTML5 CSS3 JavaScript LocalStorage

Задание для выполнения

  1. Заполните все поля формы регистрации
  2. Выберите протокол HTTPS или HTTP
  3. Выберите почтовый протокол
  4. Нажмите кнопку "Зарегистрироваться"
  5. Проверьте, появились ли ваши данные в таблице ниже
